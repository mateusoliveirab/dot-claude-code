# Agentic Framework

Small reference set for designing agents, skills, and multi-agent workflows.

Read in this order:

1. [principles.md](principles.md) - baseline rules for useful agentic systems
2. [workflow.md](workflow.md) - default operating flow for non-trivial work
3. [patterns.md](patterns.md) - reusable orchestration patterns
4. [handoffs-and-gates.md](handoffs-and-gates.md) - structured artifacts, confidence, human gates, and terminal states
5. [model-strategy.md](model-strategy.md) - route work to the cheapest model that can safely do it
6. [diagnostic-loop.md](diagnostic-loop.md) - evidence-first RCA loop with challenge and blast-radius gates
7. [design-checklist.md](design-checklist.md) - questions to answer before creating or expanding an agent

Use these docs as design references, not ceremony. A simple task should stay simple.
